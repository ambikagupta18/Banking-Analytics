This project delivers an end-to-end data analytics solution for core banking operations, designed to support risk mitigation, operational efficiency, and strategic decision-making.
Using MySQL, Tableau, Power BI, and Excel, the project analyzes loan portfolios, client behavior, branch performance, and transactional risk. Interactive dashboards provide banking stakeholders with real-time visibility into financial health, repayment risk, and fraud indicators.
